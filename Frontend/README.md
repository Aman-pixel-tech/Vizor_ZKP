# 🧩 Problem Statement 

Our solution provides a **Privacy Preserving Digital Authentication System** using **Zero Knowledge Proof (ZKP)**, where users can prove their identity without revealing their secret credentials.

---

# 🔎 What is Zero Knowledge Proof (ZKP)?

Zero Knowledge Proof is a cryptographic method that allows one party (Prover) to prove to another party (Verifier) that they know a secret — **without revealing the secret itself**.

This ensures:
- ✅ No password transmission  
- ✅ No sensitive data storage  
- ✅ Strong cryptographic authentication  
- ✅ Protection against replay attacks
